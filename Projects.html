Spending Tracker

My very first project made using Python. It is a command-line personal finance tool designed to help users understand and manage their daily and weekly spending habits. This tracker stores user-specific data in individual text files and enables long-term tracking without requiring a login system. Users can record the amount spent, mood, category (e.g., food, shopping, bills), and the time of each purchase.

Key features include:

  • Custom file creation: Generates and maintains a personalized log for each user based on their name and age.
  • Spending logs: Records purchases with timestamp, mood, and category for better emotional and financial awareness.
  • Summaries: Provides a breakdown of total and average spending by day, week, and month.
  • Most frequent category: Identifies where the user spends the most money.
  • Mood analysis: Highlights the most common mood during purchases.
  • Data cleanup: Lets users delete individual spending entries or reset their entire history.

This project was built using core Python concepts such as user-defined functions, loops, conditionals, file I/O, and formatted string output.
